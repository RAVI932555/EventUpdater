
# EventUpdater
AI-powered College Event Management System built using Spring Boot &amp; React. Students can browse &amp; apply for events, receive notifications, and get AI-based event recommendations.

EventUpdater is a full-stack platform where colleges can publish events and students can browse, apply, receive notifications, and get AI-powered event recommendations based on their interests.
